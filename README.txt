/ - Time2Learn - Eğitim Takvimi Uygulaması - / 

-> React Native (Expo + TypeScript) ile geliştirilmiş modern eğitim etkinlik yönetim uygulaması.


 Özellikler : 

    • Eğitim etkinliklerini listeleme

    • Gelişmiş arama ve filtreleme

    • Telefon/WhatsApp ile iletişim

    • Harita entegrasyonu

    • Takvim entegrasyonu (ICS)

    • Kontenjan ve ücret bilgileri

    • Online/Yüz yüze/Hibrit etkinlik desteği


Kullanılan Teknolojiler : 

    • React Native (Expo)
   
    • TypeScript

    • React Navigation

    • Redux (State Management)

    • Expo Calendar (Takvim Entegrasyonu)


Geliştirici : Eren Söğütlü


Kurulum :

    > git clone https://github.com/erensogutlu/Time2Learn.git

    > cd time2learn

    > npm install


Derleme ve Dağıtım :

    Android :
    > expo build:android -t apk

    iOS :
    > expo build:ios

    EAS :
    > eas build -p android --profile production
    > eas build -p ios --profile production

/ - Time2Learn - Educational Calendar App - /

-> A modern educational event management application developed with React Native (Expo + TypeScript).

Features:

   • Listing of training events

   • Advanced search and filtering

   • Communication via phone/WhatsApp

   • Map integration

   • Calendar integration (ICS)

   • Quota and fee information

   • Online/In-person/Hybrid event support

Technologies Used:

   • React Native (Expo)

   • TypeScript

   • React Navigation

   • Redux (State Management)

   • Expo Calendar (Calendar Integration)

Developer: Eren Söğütlü

Installation:

   > git clone https://github.com/erensogutlu/Time2Learn.git

   > cd time2learn

   > npm install

Build and Deployment:

   Android:
   > expo build:android -t apk

   iOS:
   > expo build:ios

   EAS:
   > eas build -p android --profile production
   > eas build -p ios --profile production
